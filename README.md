# How To Compile?
- Install make
- Install nasm
- Install qemu

For compilation, use:

```
make compile
```

For cleaning, use:

```
make clean
```

For testing, use:

```
make run
```

For compile and run, use:

```
make rc
```

<b>Note: Ensure you have all the necessary dependencies installed before running the commands above.</b>
